1. What is the difference between var, let, and const?
=> var, let, and const in JavaScript differ mainly in scope and reassignability. var is function-scoped (or global), hoisted, and re-declarable. let and const are block-scoped (scoped to {}), with let allowing reassignment and const prohibiting it. const requires initialization, while let and var do not.

2. What is the spread operator (...)?
=> The spread operator (...) in JavaScript is used to expand or unpack elements from an array, object, or iterable into individual elements.

3. What is the difference between map(), filter(), and forEach()?
=>The primary difference is the return value and purpose: map() transforms each element into a new array, filter() selects a subset of elements into a new array, and forEach() performs an action on each element without returning anything. None of them modify the original array by default. 

4. What is an arrow function?
=>Arrow functions, introduced in ES6, are a concise syntax for writing function expressions in JavaScript using the => operator. They offer shorter syntax—often omitting function, return, and {}—and automatically bind the this keyword to the surrounding lexical scope. They are ideal for callbacks but cannot be used as constructors

5. What are template literals?
=> Template literals in JavaScript are a way to create strings using backticks ( ) instead of single ' ' or double " " quotes.
They make it easier to insert variables and write multi-line strings.
